Naming Convention adalah aturan yang harus diikuti dalam membuat suatu variabel, class dan sebagainya agar kode terlihat lebih profesional.
Penamaan variabel dan class dalam pemrograman itu sangat penting agar terhindar dari kesalahpahaman antar programmer. Penamaan yang baik akan memudahkan programmer yang melanjutkan proyek tersebut karena penamaan yang baik bisa memberi gambaran apa yang dilakukan variabel atau class tersebut
Java menyediakan lima operator aritmatika dasar yaitu tambah (+), kurang (-), kali (\*), bagi (/) dan modulus (%). Java mendukung operator increment (++) dan decrement (--) yaitu menambahkan atau mengurangkan nilai 1 pada variabel, perhitungan pada operator ini berdasarkan posisi variabelnya yaitu prefix (variabel diakhir) dan postfix (variabel diawal) contohnya x++ atau --x. Java juga mendukung Compound Assignment Operators membuat kode menjadi lebih ringkas dengan cara menggabungkan operasi aritmatika dengan assignment (pemberian nilai ke variabel) contohnya x += 1. Java mendukung Operator Precedence yaitu menentukan aturan prioritas urutan evaluasi, kita bisa mengubah urutan evaluasi dengan tanda kurung "()". Java mendukung Type Casting dan Konversi yaitu mengubah tipe data misalnya dari integer ke desimal, ada dua jenis antara lain Widening (Implicit Conversion) yaitu konversi dari tipe data kecil ke tipe data besar dan lebih aman serta tidak akan kehilangan data sedangkan Narrowing (Explicit Casting) yaitu konversi tipe data besar ke tipe data kecil dan beresiko kehilangan data.
